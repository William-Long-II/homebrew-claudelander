# Homebrew Tap for ClaudeLander

This is the official Homebrew tap for [ClaudeLander](https://github.com/William-Long-II/claudelander), a cross-platform Claude Code session manager.

## Installation

```bash
brew tap William-Long-II/claudelander
brew install --cask claudelander
```

## Updating

```bash
brew update
brew upgrade --cask claudelander
```

## Uninstallation

```bash
brew uninstall --cask claudelander
brew untap William-Long-II/claudelander
```
